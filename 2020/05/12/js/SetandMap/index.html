Set 和 Map 数据结构

Set

ES6 提供了新的数据结构 Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。

属性和方法

  1. Set.protitype.size 返回 Set 实例成员总数
  2. Set.protitype.add(value) 给 Set 添加某个值，返回 Set 结构本身
  3. Set.protitype.delete(value) 删除某个值，返回布尔值，表示是否删除成功
  4. Set.protitype.has(value) 返回一个布尔值，表示是否为 Set 成员
  5. Set.protitype.clear() 清除所有成员，没有返回值

Map

Map 提供了”值->值”的对应

属性和方法

  1. Map.protitype.size
  2. Map.protitype.set(key,value)
  3. Map.protitype.get(key)
  4. Map.protitype.has(key)
  5. Map.protitype.delete(key)
  6. Map.protitype.clear()
