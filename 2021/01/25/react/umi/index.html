  1. model.ts

    约定式的 model 组织方式，不用手动注册 model

  • 创建一个对象并导出
  • 对象中有 4 个属性
    • namespace: string
    • state: {}
    • effects: { *函数: Effect }
      • 函数有两个参数
      • 函数({params},{call,put})
    • reducers: { *函数: Reducer }
