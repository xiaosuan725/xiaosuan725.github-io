TodoList(cli版)

之前做了todolist简单版，组件版，这次使用脚手架完成todolist，使用到了setup新增的钩子函数

setup钩子函数

  1. vue3提供的组合api只能在一个地方使用： 入口【setup】
  2. 特点
    1. 它是组合api统一入口
    2. this是undefined
    3. setup的执行在组件创建前 beforeCreate前执行
  3. 参数
    1. props 属性
    2. context 上下文对象
      1. emit this.$emit context.emit
  4. 返回值： 一定是一个对象
    1. 数据
    2. 方法

组合api

  1. 数据定义
    1. reactive
      1. 定义内部数据
      2. 修改这个数据代码必须在setup内部运行 【类似： 私有】
    2. ref
      1. 定义数据不仅内部可以使用，外部也可以使用
      2. 它的数据必须通过value才能拿到
  2. 把两者结合起来一起用【 toRefs 】
  3. isRef
  4. computed
    1. 不可修改的computed const reverseMsg = computed(() => {})
    2. 可修改的computed const reverseMsg = computed({get () {},set () {}})
  5. watch
    1. 监听单一数据源
    2. 多个数据源
  6. 生命周期
    1. onMounted
    2. onUpdated
    3. onUnmounted
